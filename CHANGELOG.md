# Changelog

All notable changes to this project will be documented in this file.

## [4.31] - 2026-04-14

### Added
- 快捷悬浮输入窗：全局热键（默认 ⌃⌥Space，可在设置中自定义）在任何前台 app 中呼出
  - 两阶段输入：输入文字 → ↑↓ 选择「新任务」或已有任务 → Enter 确认；选已有任务会 pin 住并清空输入让你填进展
  - 候选列表实时过滤任务名
  - Pin 状态下显示该任务最近 5 条进展历史
  - Esc / Backspace（空输入）/ X 按钮：两段回退（pin → 取消 pin；否则关窗）
- 菜单栏图标：左键呼出悬浮窗，右键菜单（打开主窗口 / 设置 / 退出）
- 关闭主窗口后 app 驻留后台，菜单栏图标与全局热键继续工作
- 设置 →「快捷键」标签：录制自定义全局热键，冲突自动提示
- 分区循环快捷键：主界面与悬浮窗均支持 ⌘] 下一个 / ⌘[ 上一个

### Changed
- 悬浮窗完全沿用主界面主题系统（elevated/bg/border/accent + t1/t2/t3），跟随主题切换
